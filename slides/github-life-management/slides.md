# GitHubで日常を管理してみた

## Obsidian + GitHub で"どこでもメモ"環境をつくる

---

## きっかけ

「**GitHubで人生を管理する**」(Zenn / hand_dot さん)

- GitHub Issue でタスク管理
- Markdown で日記を書く
- 仕事で使い慣れたツールで生活も管理する

> エンジニアなら、普段使っている道具をそのまま活かせばいい

---

## 自分の課題

- 日々の **TODO** ややりたいことを一箇所にまとめたい
- PC でもスマホでも同じメモにアクセスしたい
- でも専用アプリに縛られたくない

---

## やっていること

**Obsidian + GitHub** で Markdown 直接編集

- Obsidian で `.md` ファイルを管理
- GitHub リポジトリで同期 → **どのデバイスからでも編集可能**
- テキストファイルだから、どのアプリでも開ける

---

## なぜ Obsidian？

| 特徴 | メリット |
|---|---|
| テキストファイルで保存 | 特定のアプリに依存しない・いつでも乗り換え可能 |
| ネットなしでも使える | オフラインでも編集できる |
| 拡張機能が豊富 | GitHub との自動同期もワンタッチ |
| どの端末でも使える | PC / スマホ / タブレット対応 |

---

## やってみたいこと

**Claude Code** を使ってもっと便利にしたい

- メモの自動整理・要約
- TODO の優先度提案
- 過去のメモから関連情報を検索
- 定期レビューの自動生成

**エンジニアの道具をフル活用して、日常も最適化する**
